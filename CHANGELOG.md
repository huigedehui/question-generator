# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-11

### Added
- Web interface for question generation
- Support for multiple AI models (Zhipu, OpenAI, DeepSeek)
- Document upload (docx, txt)
- Multiple question types (single choice, multiple choice, true/false, essay)
- Pause and save functionality
- Real-time progress display
- Excel export
- API connection testing
- Command-line interface

### Features
- Semantic text chunking
- Question validation and cleaning
- Random sampling by question type
- Template column mapping
- Progress tracking
- Question caching

## [0.1.0] - 2026-05-11

### Added
- Initial release
- Basic module structure
- Word document processing
- AI API client with retry logic
