# 快速入门指南

## 环境要求

- Python 3.8+
- Windows / Linux / macOS

## 安装步骤

### 1. 克隆项目

```bash
git clone <项目地址>
cd question_generator
```

### 2. 创建虚拟环境（推荐）

```bash
python -m venv venv
# Windows
venv\Scripts\activate
# Linux/macOS
source venv/bin/activate
```

### 3. 安装依赖

```bash
pip install -r requirements.txt
```

### 4. 获取 API Key

推荐使用智谱 AI（免费模型）：

1. 访问 https://www.bigmodel.cn
2. 注册账号
3. 控制台 → API Keys → 创建密钥
4. 复制密钥备用

### 5. 启动服务

```bash
python app.py
```

浏览器访问：http://localhost:5000

## 使用流程

### 1. API 配置

1. 填入 API Key
2. 选择 API 提供商（默认智谱AI）
3. 选择模型（推荐 GLM-4-Flash-250414）
4. 点击"测试连接"验证

### 2. 上传文档

支持格式：
- Word 文档 (.docx)
- 文本文件 (.txt)

### 3. 设置参数

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 单选题 | 生成单选题数量 | 150 |
| 多选题 | 生成多选题数量 | 90 |
| 判断题 | 生成判断题数量 | 90 |
| 问答题 | 生成问答题数量 | 20 |
| 切片大小 | 每块字符数 | 800 |
| 难易度 | 题目难度等级 | 一般 |

### 4. 开始生成

点击"开始生成"，日志区域实时显示进度。

### 5. 暂停与保存

- **暂停并保存**：停止生成，保存已生成的题目
- **保存当前**：随时保存当前已生成的题目

## 导出文件

生成完成后，点击"下载 Excel"获取题目文件。

## 故障排除

### 连接失败

1. 检查 API Key 是否正确
2. 检查网络是否正常
3. 确认 API 余额充足

### 生成数量少

1. 减小切片大小（建议 500-600）
2. 上传更多文档
3. 文档内容是否足够丰富

### Excel 打开报错

确保安装了 Microsoft Excel 或使用 WPS 打开。
